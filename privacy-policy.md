# Privacy Policy – PokéFarm Field IV Viewer

**Last Updated:** March 12, 2026

## Overview

PokéFarm Field IV Viewer is a Chrome browser extension that enhances the PokéFarm Fields page by displaying Pokémon Individual Values (IVs) directly in the tooltip when hovering over a Pokémon.

The extension is designed to operate entirely within the user's browser and prioritizes user privacy.

This extension **does not collect, store, transmit, or sell personal data**.

---

# Chrome Web Store Data Disclosure

The Chrome Web Store requires developers to disclose how extensions handle user data.

PokéFarm Field IV Viewer provides the following guarantees:

### Data Collection

This extension **does not collect any user data**.

The extension does **not collect or process**:

- Personal information
- Authentication information
- Financial or payment data
- Health information
- Location data
- Browsing history
- User activity
- Website content unrelated to PokéFarm
- Device identifiers
- Analytics data

---

### Data Usage

The extension only reads publicly available data from pages on: [PokéFarm Q](https://pokefarm.com)

Specifically, it retrieves Pokémon summary pages in order to extract IV values and display them within the existing PokéFarm tooltip interface.

All data processing occurs **locally within the browser**.

---

### Data Storage

The extension may temporarily store Pokémon IV values **in memory** to avoid repeated network requests and improve performance.

This cache:

- Exists only during the active browser session
- Is never written to persistent storage
- Is automatically cleared when the page reloads or the browser is closed

No user data is permanently stored.

---

### Data Sharing

The extension **does not share any data** with:

- Third-party services
- Analytics providers
- Advertisers
- External servers
- The extension developer

---

### Data Sale

The developer **does not sell or transfer user data to third parties**.

---

### Data Usage Limitations

All functionality of the extension is strictly limited to improving the user interface of the PokéFarm website.

Data accessed by the extension is used **solely for the purpose of displaying Pokémon IV values in tooltips**.

---

# Permissions

The extension requests access to the following host permission: [PokéFarm Q](https://pokefarm.com)

This permission is required in order to:

- Detect Pokémon elements on the PokéFarm Fields page
- Retrieve Pokémon summary pages
- Extract IV data
- Display IV values within tooltips

The extension **does not access any other websites**.

---

# Third-Party Services

This extension does **not use any third-party libraries, APIs, analytics platforms, or tracking tools**.

---

# Security

All extension functionality runs locally in the browser and no external services are involved. As a result, no user data leaves the user's device.
